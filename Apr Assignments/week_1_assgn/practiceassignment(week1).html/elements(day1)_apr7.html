HTML

  • html is markup and presenting language which is used to structure the content in web page, using the elements provided. Attributes are used to define the properties of html tags which gives you extra information about tags . Attributes are always be in the key value pair format and always configured in the opening Tags.

  • HTML - stands for Hyper Text Markup Language. Some Elements in HTML are Heading elements,Paragraph Elements,division Elementsetc.,

  • Elements in the HTML are used to structure the content in web page. Where as Tags are the formats of the elements.

Examples:
  1. div
  2. img
  3. span
  4. p

CSS

  • CSS-Stands for Cascading styling sheet.

  • CSS is used to style the content of the web page such as color,fontsize,fontstyle etc.,

CSS are of three types :-
  1. Inline CSS
  2. Internal CSS
  3. External CSS
